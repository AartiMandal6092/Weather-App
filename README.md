# Weather-App

This Python project fetches the **current weather** of any city and can **speak it out loud** using your computer.
**Weather Speaker** is a simple Python program that fetches the **current weather** of any city and reads it out loud using your computer.  
This project is beginner-friendly and a fun way to learn Python, APIs, and text-to-speech.


---

## Features

- Shows **current temperature**, **wind speed**, and **wind direction**
- Converts the weather info into **speech**
- Easy to use: just run the Python file
